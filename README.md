# lewis
Drawing Lewis Structure on a Canvas
This is a free form chemical drawing canvs design to draw Lewis sructure and also drawing 3D projection on a 2D canvs. For ionic species, one can also add a bracket and charge.
INSTRUCTIONS
1) Add atom from a list or from periodic table
2) Add bond type (single, double, or triple) to connect another atom.
3) Click on the atom to add lone-pair electrons
4) To change bonglength, click on the atom and move it to the desire location
5) to change the location of the lone-pair, click on the atom first, then move the lone pair round the atom
